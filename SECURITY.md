# Security Policy

## Scope

This project is a cybersecurity education toolkit built on Digispark ATtiny85 hardware. It is intended for:

- College and university exhibitions
- Capture The Flag (CTF) competitions
- Authorized penetration testing demonstrations
- Personal security research on devices you own

---

## Responsible Use

By using this toolkit you agree to:

- Only deploy on devices you own or have explicit written permission to test
- Not distribute modified payloads designed for unauthorized access
- Not use this tool against networks, systems or individuals without consent

Unauthorized use may violate:
- IT Act 2000 (India)
- Computer Fraud and Abuse Act (USA)
- Computer Misuse Act (UK)
- Similar laws in your jurisdiction

---

## Reporting a Vulnerability

If you discover a security issue in this project itself (such as a flaw in the payload cleanup logic or a way the toolkit could be misused more broadly than intended), please:

1. Do **not** open a public GitHub issue
2. Email the maintainer directly via GitHub profile contact
3. Include a clear description of the issue and potential impact

We aim to respond within 7 days and will coordinate a fix and disclosure timeline with you.

---

## Disclosure Policy

- Security issues will be fixed in a patch release
- Credit will be given to the reporter unless anonymity is requested
- We follow responsible disclosure: fixes ship before public details are shared
